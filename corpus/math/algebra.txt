# ALGEBRA FOUNDATION
# Variables, equations, and algebraic operations

# Variables
VARIABLE → REPRESENT → UNKNOWN
X → VARIABLE → SOLVE
Y → VARIABLE → SOLVE
EQUATION → VARIABLE → SOLVE

# Equations
EQUATION → LEFT_SIDE → EQUALS → RIGHT_SIDE
EQUATION → SOLVE → VARIABLE → VALUE
LINEAR → EQUATION → SOLVE
QUADRATIC → EQUATION → SOLVE

# Operations on equations
EQUATION → ADD → BOTH_SIDES → EQUIVALENT
EQUATION → SUBTRACT → BOTH_SIDES → EQUIVALENT
EQUATION → MULTIPLY → BOTH_SIDES → EQUIVALENT
EQUATION → DIVIDE → BOTH_SIDES → EQUIVALENT

# Functions
FUNCTION → INPUT → OUTPUT
FUNCTION → DOMAIN → RANGE
FUNCTION → EVALUATE → VALUE
LINEAR → FUNCTION → GRAPH
QUADRATIC → FUNCTION → GRAPH

# Polynomials

POLYNOMIAL → EXPAND → EXPRESSION

# Systems of equations
EQUATION → SYSTEM → SOLVE → SOLUTION
TWO_EQUATIONS → TWO_VARIABLES → SOLVE
