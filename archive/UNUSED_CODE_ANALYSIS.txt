========================================
PATTERN STRUCTURE UNDERSTANDING
========================================

Patterns should be:
- PATTERN_ROOT nodes (kind = NODE_KIND_PATTERN_ROOT)
- Connected to BLANK nodes (kind = NODE_KIND_BLANK) via edges
- Connected to DATA nodes (kind = NODE_KIND_DATA) via edges
- Connected to other PATTERN nodes via edges
- All via universal edges (EDGE_FLAG_PATTERN | EDGE_FLAG_BIND)

Pattern structure:
  PATTERN_ROOT
    ↙  ↓  ↘
  BLANK nodes (variables)
  DATA nodes (concrete values)
  PATTERN nodes (nested patterns)
    ↙  ↓  ↘
  CHANNEL nodes (I/O)

Edges are universal - they connect any node to any node.
